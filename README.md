# currencyExchange
task from FocusStart

Выполнены задания 1,2 и частично 3.

Файл.json загружается с сервера, десериализуется в объекты java, данные вносятся в SQLiteDatabase 
и оттображаются на экране внутри фрагмента посредством recyclerView.

Кнопка на тулбаре инициирует загрузку данных с сервера.

Клик по наименовании валюты из списка открывает всплывающее окно - интерфейс для конвертации указанной суммы рублей в выбранную валюту.
